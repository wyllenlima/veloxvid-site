02 Descrição do serviço

O VeloxVid é um aplicativo desktop para Windows que oferece ferramentas de produção de conteúdo com inteligência artificial processada localmente no dispositivo do usuário, incluindo:

  • Transcrição de áudio e vídeo com IA local (faster-whisper)
  • Geração de roteiros e prompts para VEO3
  • Automação de envio de prompts para a API VEO3
  • Geração de títulos, descrições e sugestões de thumbnails
  • Conversão de arquivos no formato JSON

05 Processamento local e privacidade

O VeloxVid processa arquivos de áudio e vídeo exclusivamente no dispositivo do usuário. Os arquivos de mídia não são enviados para servidores externos durante o processo de transcrição.

Dados de autenticação e licenciamento são processados de forma segura por meio de infraestrutura terceirizada. Para mais detalhes, consulte nossa Política de Privacidade.
